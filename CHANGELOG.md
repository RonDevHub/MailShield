# Changelog

### Fixed
- The CAPTCHA integration has been updated and can no longer be bypassed.
- The light/dark mode toggle has been redesigned and is now located in the page header, so there are no visual disruptions on the mobile site

### Added
- SQLite (default, zero config), PostgreSQL (recommended for production) Switch via environment variables.
- Easily switch from an SQLite database to PostgreSQL (migration can now be performed using JSON)
